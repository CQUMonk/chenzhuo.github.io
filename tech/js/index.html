# JavaScript

对于一门编程语言来说，都是通过一定的语法和词法，来表达一定的语义，从而操作运行时（数据结构+算法）。 JavaScript作为Web端开发的语言，江湖上常以高度的动态性和灵活性而著称，与Java、C++等传统编程语言有很多不同的地方。
